Line Bot

# require node >= v8.x.x
check your nodejs version
`node -v`


How to run ?
------
- `pkg install git -y`
- `pkg install nodejs -y`
- `git clone https://github.com/fianchiks/LineBot.git`
- `cd LineBot && npm install`
- `pkg install nodejs-current`
- `npm start`
